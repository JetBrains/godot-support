Reports sections that appear out of order in TSCN and TRES files.

Godot requires sections in the following order:

  1. [gd_scene] or [gd_resource] - File descriptor
  2. [ext_resource] - External resources
  3. [sub_resource] - Internal resources
  4. [node] - Nodes
  5. [connection] - Connections

Example of incorrect order:

[gd_scene format=3]
[node name="Root" type="Node"]
[ext_resource path="res://script.gd" type="Script" id=1]  ; ERROR: ext_resource after node

Example of correct order:

[gd_scene format=3]
[ext_resource path="res://script.gd" type="Script" id=1]
[node name="Root" type="Node"]
